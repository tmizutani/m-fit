\section{Detecção de transições\label{carrega_video}}

\hspace{1.25cm}Nesta seção detalharemos os passos necessários para
se detectar as transições de um Vídeo.

\hspace{0.65cm}Para esta funcionalidade temos dois caminhos:

\begin{enumerate}
\item{À partir do carregamento do Vídeo}

\subitem{ Ao chegar no passo 4 do processo de carregamento de Vídeo
(Seção \ref{carrega_video}), existe a opção de aceitar a detecção de
todas as transições. Para isso, basta clicar em ``OK'' quando o
sistema questionar sobre a detecção. Com isso o processo de detecção
de todas as transições se iniciará. ( Figura \ref{img:carregar4} )}

\item{Após o carregamento do Vídeo}

\subitem{ Após realizar todos os passos do processo de carregamento do Vídeo (Seção \ref{carrega_video}),
os botões de detecção de transição estarão habilitados. Escolha a detecção que deseja e pressione o botão (Figura \ref{img:carregar5} )}

\item{ Após escolher um dos passos acima, o sistema realizará a detecção de transições, com isso as transições ficarão delimitadas
na Timeline, e a lista de transições será preenchida (Figura
\ref{img:transicoes1}).}

\end{enumerate}

\section{Configurações da Detecção \label{config_detecta}}

\hspace*{1.25cm}Para realizar a detecção de transições, o usuário
tem como opção alterar algumas configurações com o intuito de
melhorar a performance da detecção.

\hspace{0.65cm}A Figura \ref{img:configuracoes} mostra a janela de
configurações que é mostrada quando o usuário acessa o menu de
configurações.

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.8\linewidth]{imagens/configuracoes.png}
 \caption{Janela de configurações.}
 Fonte: Autor.
 \label{img:configuracoes}
\end{figure}

\hspace*{0.65cm}No canto esquerdo desta janela, encontra-se um menu
que dá acesso às configurações específicas pra cada tipo de
transição. No entanto só estão disponíveis nesta versão, as
configurações para detecção do tipo corte.

\hspace*{0.65cm}Cada item é explicado a seguir:

\begin{enumerate}

\item{\textbf{Porcentagem:}
Aqui é possível alterar o valor do limiar para definição de quais
bordas geradas através da aplicação do Filtro de Canny sobre o Ritmo
Visual serão consideradas cortes. O valor aqui definido corresponde
à porcentagem da altura do Frame. Portanto quanto maior for esta
porcentagem, maiores deverão ser as bordas para que sejam
consideradas cortes. Recomenda-se não utilizar valores muito baixos
para que a quantidade de falsas detecções aumente.}

\item{\textbf{Limiar Mínimo:}
Este limiar corresponde ao limiar mais baixo para o processo de
limiarização dupla, ou histerese. Quanto mais baixo for este limiar,
maior será o número de bordas detectadas, causando um aumento do
número de falsas detecções. No entanto, para cada tipo de vídeo há
uma configuração que irá gerar resultados mais satisfatórios.}

\item{\textbf{Limiar Máximo:}
Este limiar corresponde ao limiar mais alto para o processo de
limiarização dupla, ou histerese. Seu comportamento é semelhante ao
do limiar mínimo, sendo que para cada tipo de vídeo há uma
configuração que gera melhores resultados.}

\end{enumerate}
