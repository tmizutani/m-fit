\section{Arquivo \label{menu_arquivo}}

\hspace{1.25cm}Neste menu existem as seguintes opções:

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.5\linewidth]{imagens/menu_arquivo.png}
 \caption{Menu Arquivo.}
 Fonte: Autor.
 \label{img:menu_arquivo}
\end{figure}

\begin{enumerate}

\item{\textbf{Novo Projeto     :} Cria um novo projeto.}
\item{\textbf{Abrir Projeto    :} Abre um projeto já existente. São aceitos somente arquivos com a extensão MFIT. }
\item{\textbf{Carregar Video   :} Carrega o Vídeo desejado. Após o carregamento, a Timeline será criada, e várias opções
estarão habilitadas para uso. São aceitos somente arquivos com a extensão AVI.}
\item{\textbf{Renderizar Video :} Renderiza o Vídeo corrente, aplicando nele todos os efeitos definidos pelo usuário. Esta opção só estará habilitada no momento em que um Vídeo for carregado.}
\item{\textbf{Salvar Como      :} Salva o projeto corrente em arquivo. Todas as alterações do usuário ficarão salvas para
uso futuro. Esta funcionalidade requisita o nome de projeto ao usuário. Esta opção só estará habilitada no momento em que um Vídeo for carregado.}
\item{\textbf{Salvar           :} Salva o projeto corrente, caso um nome já esteja definido, o projeto é salvo sem requisitar um nome de projeto. Esta opção só estará habilitada no momento em que um Vídeo for carregado.}
\item{\textbf{Arquivos Recentes:} O espaço entre os 2 limitadores é reservado para os últimos arquivos abertos, sejam eles de
vídeo ou de projeto. }
\item{\textbf{Sair             :} Fecha o MFIT.}

\end{enumerate}

\section{Detecção \label{menu_deteccao}}

\hspace{1.25cm}Este menu só tem suas opções habilitadas para uso se existir um Vídeo carregado.
Neste menu existem as seguintes opções:

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.2\linewidth]{imagens/menu_deteccao.png}
 \caption{Menu Detecção.}
 Fonte: Autor.
 \label{img:menu_deteccao}
\end{figure}

\begin{enumerate}

\item{\textbf{Todas   :} Inicia o processo de detecção de todas as transições: Corte, Fade e Dissolve. }
\item{\textbf{Corte   :} Inicia o processo de detecção de transições do tipo Corte. }
\item{\textbf{Fade    :} Inicia o processo de detecção de transições do tipo Fade. }
\item{\textbf{Dissolve:} Inicia o processo de detecção de transições do tipo Dissolve.}

\end{enumerate}

\section{Barra de Botões \label{barra_de_botoes}}

\hspace{1.25cm}Logo abaixo dos menus do sistema se encontra a barra de botões.
Esta barra é uma extensão de todos os menus, sendo que cada botão, tem uma entrada correspondente em algum dos menus.
Esta barra proporciona fácil acesso a opções dos menus existentes.

\begin{figure}[h|top]
 \centering
 \includegraphics[width=0.5\linewidth]{imagens/barra_botoes.png}
 \caption{Barra de Botões.}
 Fonte: Autor.
 \label{img:barra_botoes}
\end{figure}

\hspace{0.65cm}Da esquerda para a direita, temos:

\begin{enumerate}

\item{\textbf{Novo Projeto}}
\item{\textbf{Abrir Projeto}}
\item{\textbf{Carregar Vídeo}}
\item{\textbf{Salvar}}
\item{\textbf{Renderizar Vídeo}}
\item{\textbf{Carregar Vídeo}}

\item{\textbf{Detecção de transição: Todas}}
\item{\textbf{Detecção de transição: Corte}}
\item{\textbf{Detecção de transição: Fade}}
\item{\textbf{Detecção de transição: Dissolve}}

\item{\textbf{Configurações: Detecção}}

\end{enumerate}

\hspace{0.65cm}Note que os ícones destes botões são os mesmos
utilizados nos menus.
